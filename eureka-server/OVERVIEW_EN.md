# Eureka Server - Recruiter Overview (EN)

`eureka-server` is the discovery backbone that allows services to find each other dynamically.

## Recruiter snapshot

- Pattern: service discovery for distributed systems
- Outcome: less hardcoded infrastructure and easier environment changes
- Technical signal: gateway/services communicate by service name, not fixed host/port

## Business value

This module improves deployment flexibility and keeps inter-service communication maintainable as the system grows.
