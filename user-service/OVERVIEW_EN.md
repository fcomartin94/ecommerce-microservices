# User Service - Recruiter Overview (EN)

`user-service` handles signup/login and issues JWT tokens for authenticated access.

## Recruiter snapshot

- Pattern: stateless authentication in a microservices backend
- Security signal: password hashing plus token-based auth
- Product signal: reusable authentication boundary for client apps

## Business value

This module provides a production-style auth foundation that protects APIs while keeping integration straightforward.
